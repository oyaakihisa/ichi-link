# 要求内容

## 概要

ichi-link MVPの全機能を実装する。位置情報の入力から地図起動までの一連のフローを完成させる。

## 背景

PRDで定義されたP0機能（7つ）を、技術的な依存関係を考慮した順序で実装する必要がある。
PRDの優先順位は「ビジネス的な必要性」であり、「実装順序」とは異なる。

## PRD機能要件と実装順序の対応

### PRDのP0機能（ビジネス優先度順）

| # | 機能名 | PRD優先度 | 実装フェーズ |
|---|--------|----------|-------------|
| 1 | 位置情報の入力受付 | P0 | フェーズ3 |
| 2 | 入力内容の自動判定 | P0 | フェーズ1 |
| 3 | 測地系の選択・推定 | P0 | フェーズ1 |
| 4 | 出力の標準化 | P0 | フェーズ1 |
| 5 | 地図URL生成・起動 | P0 | フェーズ1 |
| 6 | 安全確認UI（警告表示） | P0 | フェーズ1, 3 |
| 7 | コピー操作の最適化 | P0 | フェーズ3 |

### 実装順序の理由

```
フェーズ1: サービス層（ビジネスロジック）
  ↓ UIはサービス層に依存
フェーズ2: プロジェクトセットアップ
  ↓ UIコンポーネントはプロジェクト基盤に依存
フェーズ3: UIレイヤー
  ↓ 統合にはUI + サービスが必要
フェーズ4: 統合・テスト
```

## 実装対象の機能

### 1. InputParser（入力パーサー）
- 入力文字列の種別を自動判定（十進度/度分秒/住所/不明）
- 各形式の座標をパース
- 判定の確信度を計算

### 2. CoordinateConverter（座標変換エンジン）
- 度分秒 ↔ 十進度 の相互変換
- 座標の正規化（小数点以下6桁）

### 3. DatumTransformer（測地系変換）
- WGS84 ↔ JGD2011 の相互変換
- 旧日本測地系（Tokyo Datum）からの変換
- proj4jsを使用

### 4. ValidationService（検証・警告生成）
- 座標が日本国内かどうかの検証
- 緯度経度の順番の妥当性検証
- 警告メッセージの生成

### 5. MapUrlGenerator（地図URL生成）
- Google Maps URL生成
- Yahoo!地図 URL生成
- Apple Maps URL生成
- 地理院地図 URL生成

### 6. UIコンポーネント群
- LocationInput（入力欄）
- ConversionResult（結果表示）
- WarningDisplay（警告表示）
- MapButtons（地図ボタン）
- CopyButton（コピーボタン）

### 7. 統合サービス
- ConversionService（全体のオーケストレーション）
- useConversion フック（React統合）

## 受け入れ条件

### サービス層
- [ ] InputParserが十進度座標を正しく判定・パースできる
- [ ] InputParserが度分秒座標を正しく判定・パースできる
- [ ] CoordinateConverterが度分秒↔十進度を正確に変換できる
- [ ] DatumTransformerがWGS84↔JGD2011を正確に変換できる
- [ ] ValidationServiceが日本国外の座標を検出できる
- [ ] ValidationServiceが緯度経度の入れ替わりを検出できる
- [ ] MapUrlGeneratorが4つの地図サービスのURLを生成できる

### UIレイヤー
- [ ] 単一の入力欄から座標を入力できる
- [ ] 変換ボタンで変換を実行できる
- [ ] 変換結果（WGS84, JGD2011）が表示される
- [ ] 警告がある場合に警告が表示される
- [ ] 4つの地図ボタンが表示され、クリックで地図が開く
- [ ] コピーボタンで座標をコピーできる

### 統合
- [ ] 入力から地図起動までの一連のフローが動作する
- [ ] すべてのユニットテストが通る
- [ ] ビルドが成功する

## 成功指標

- 主要入力パターン（十進度、度分秒）の変換成功率: 100%
- 入力から結果表示まで: 500ms以内
- 地図URL生成: 100ms以内

## スコープ外

以下はこのフェーズでは実装しません:

- 住所入力のジオコーディング（外部API必要）
- 逆ジオコーディング（座標→住所）
- 変換履歴の保存（LocalStorage）
- モバイル最適化の詳細調整
- E2Eテスト
