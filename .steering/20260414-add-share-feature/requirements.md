# 要求内容

## 概要

変換結果をLINEやその他のアプリで簡単に共有できる機能を追加する。

## 背景

現場で位置情報を変換した後、その結果を同僚やチームメンバーに共有したいニーズがある。
現在は各座標や地図URLを個別にコピーする必要があり、LINEなどで複数人に共有する際に手間がかかる。
日本では消防・救急の現場連絡にLINEが広く使われているため、LINEでの共有を最優先で実装する。

## 実装対象の機能

### 1. LINEで共有ボタン

- LINE Share URLを使用してワンタップでLINEアプリを開き、友達/グループに共有できる
- 共有内容: 変換結果（WGS84座標）とGoogle MapsのURL
- PC/モバイル両対応

### 2. 汎用共有ボタン（Web Share API）

- ブラウザのネイティブ共有機能を使用
- LINE以外のアプリ（メール、Slack、他のメッセンジャー等）でも共有可能
- Web Share API未対応ブラウザでは非表示

## 受け入れ条件

### LINEで共有ボタン

- [ ] 「LINEで共有」ボタンをクリックするとLINEアプリが開く
- [ ] 共有テキストにWGS84座標が含まれる
- [ ] 共有テキストにGoogle MapsのURLが含まれる
- [ ] PCでもモバイルでも動作する

### 汎用共有ボタン

- [ ] 「共有」ボタンをクリックするとOS/ブラウザのネイティブ共有シートが開く
- [ ] Web Share API未対応ブラウザでは「共有」ボタンが非表示になる
- [ ] 共有内容にWGS84座標とGoogle MapsのURLが含まれる

### UI配置

- [ ] 共有ボタンは「地図で開く」セクションの下に配置する
- [ ] ボタンは視覚的に分かりやすいデザインにする
- [ ] モバイルでもタップしやすいサイズ（44x44px以上）

## 成功指標

- 変換完了から共有完了まで3タップ以内
- ボタンの視認性が高く、初見でも使い方が分かる

## スコープ外

以下はこのフェーズでは実装しません:

- QRコード生成による共有
- 共有用の短縮URL生成
- 共有履歴の保存
- カスタム共有テンプレートの設定
