# 要求定義: 地図スタイル切り替え機能

## 概要

MapViewに衛星画像への切り替え機能を追加する。

## 要求

### 機能要求

1. 地図スタイルを切り替えられる（標準地図 / 衛星画像）
2. 右上にスタイル切り替えボタンを配置する
3. 衛星画像はラベル付き（satellite-streets-v12）を使用する
4. スタイル切り替え後も日本語ラベル設定を維持する

### 非機能要求

- ボタンはタッチしやすいサイズ（最小44x44px）
- 切り替え時の待機時間を最小限に
- 既存のナビゲーションコントロールと干渉しない配置

## スコープ外

- 3つ以上のスタイル選択（将来拡張として検討可能）
- スタイル選択の永続化（リロードで標準に戻る）
