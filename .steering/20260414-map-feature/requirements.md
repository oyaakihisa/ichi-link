# 要求内容

## 概要

タブナビゲーションとMapbox地図機能を追加し、地図上の長押しで位置情報を取得・共有できるようにする。

## 背景

現在の変換ツールに加えて、地図から直接位置情報を取得したいユーザーニーズに対応する。
緊急対応の現場では、住所や座標が分からない場合でも、地図上で場所を特定して位置情報を共有する必要がある。

## 実装対象の機能

### 1. タブナビゲーション
- 画面上部に「変換ツール」「マップ」タブを表示
- タブ切り替えで対応する画面を表示
- 選択中タブの視覚的フィードバック

### 2. マップ表示
- Mapbox GL JS（Zenrinデータ使用）による地図表示
- 日本の建物情報が正確に表示される
- ピンチ・ドラッグ操作で拡大縮小・移動

### 3. 長押しピン機能
- マップ上を長押し（約500ms）でピンを設置
- ピン位置から逆ジオコーディングで住所を取得
- WGS84座標とTokyo Datum座標を計算

### 4. 位置情報スライドパネル
- 下からスライドして表示されるボトムシート
- 共有ボタン（LINE共有、Web Share API）
- 住所、WGS84座標、Tokyo Datum座標の表示
- 各座標値のコピー機能

## 受け入れ条件

### タブナビゲーション
- [ ] 画面上部に「変換ツール」「マップ」タブを表示する
- [ ] タブをタップすると対応する画面に切り替わる
- [ ] 現在選択中のタブが視覚的に分かる（アンダーライン）
- [ ] タブ切り替え時にスムーズなトランジションがある

### マップ表示
- [ ] Mapbox GL JSによるマップが表示される
- [ ] 初期表示は日本全体が見えるズームレベル
- [ ] ピンチ操作で拡大縮小できる
- [ ] ドラッグ操作で移動できる

### 長押しピン機能
- [ ] マップ上を長押し（約500ms）するとピンが立つ
- [ ] ピンを立てると下からスライドパネルが表示される
- [ ] 既存のピンがある場合、新しいピンで置き換わる
- [ ] ピン位置から逆ジオコーディングで住所を取得する
- [ ] WGS84座標とTokyo Datum座標を計算・表示する

### 位置情報スライドパネル
- [ ] 下からスライドして表示されるパネル
- [ ] パネル上部に「LINEで共有」「共有」ボタンを配置
- [ ] 住所、WGS84座標、Tokyo Datum座標を表示
- [ ] 各座標値を個別にコピーできる
- [ ] パネルを閉じることができる

## 成功指標

- マップ読み込み時間: 初回3秒以内
- ピン反応時間: 長押し検出後100ms以内
- 逆ジオコーディング: 1秒以内
- 地図操作: 60fps以上

## スコープ外

以下はこのフェーズでは実装しません:

- 現在地取得機能（Geolocation API）
- オフラインマップキャッシュ
- 複数ピンの設置
- ピン位置の履歴保存
- 地図スタイルの切り替え
