# 修正要求

## 修正対象

- 機能名: 地図長押し時のスライドパネルのコピーボタン
- 関連ファイル: `components/map/SlidePanel.tsx`

## 問題の説明

地図を長押しした際に表示されるスライドパネルにおいて、すべての情報行（住所、WGS84座標、Tokyo Datum座標、Google Maps URL）に個別の「コピー」ボタンが付いている。また、全情報をまとめてコピーするボタンの表記が「全部コピー」となっている。

## 現在の動作

- 住所行に「コピー」ボタンあり
- WGS84座標行に「コピー」ボタンあり
- Tokyo Datum座標行に「コピー」ボタンあり
- Google Maps URL行に「コピー」ボタンあり
- 全情報コピーボタンに「全部コピー」と表示

## 期待する動作

- 各行の個別コピーボタンを削除
- 全情報コピーボタンのみを残し、表記を「コピー」に変更

## 修正アプローチ

1. `SlidePanel.tsx`から各行に付いている`<CopyButton>`コンポーネントの呼び出しを削除
2. 「全部コピー」ボタンの表記を「コピー」に変更

## 影響範囲

- `components/map/SlidePanel.tsx` - 直接修正対象

## 受け入れ条件

- [ ] 地図長押し時のスライドパネルで、住所行にコピーボタンがないこと
- [ ] WGS84座標行にコピーボタンがないこと
- [ ] Tokyo Datum座標行にコピーボタンがないこと
- [ ] Google Maps URL行にコピーボタンがないこと
- [ ] 全情報コピーボタンの表記が「コピー」になっていること
- [ ] 「コピー」ボタンをクリックすると全情報がコピーされること

## テスト計画

- npm test、npm run lint、npm run typecheckでエラーがないことを確認

## スコープ外

以下はこの修正では対応しません:

- POIモードのコピーボタン（長押しモードとは別のUIフロー）
- 変換結果モードのコピーボタン（要件では長押し時のみ言及）
- CopyButtonコンポーネント自体の修正
